# 🏘️ 永不纳什小镇 (UNASH-TOWN)

**多智能体A股交易模拟系统 - 开放式策略探索**

一个模拟小镇居民进行股票交易的系统。每个居民都有独特的初始倾向，通过自主探索寻找最优交易策略，在交易反馈中不断调整风格，最终涌现出个性化的交易行为。

## 🎮 核心特性

### 开放式策略探索

智能体不是执行硬编码的交易规则，而是：

1. **自主探索** - 尝试不同的交易策略，观察效果
2. **反馈学习** - 根据交易盈亏调整策略置信度
3. **风格演化** - 风险偏好、仓位大小等参数动态调整
4. **个性涌现** - 不同初始倾向导致不同风格发展

### 8种交易策略

智能体可以探索和切换的策略：

| 策略 | 描述 | 触发条件 |
|------|------|----------|
| Value | 价值投资 | 低估买入，高估卖出 |
| Momentum | 动量交易 | 追涨杀跌 |
| Mean Reversion | 均值回归 | 超买卖，超买卖 |
| Breakout | 突破交易 | 突破阻力买入 |
| Scalping | 短线交易 | 快进快出 |
| Swing | 波段交易 | 持仓数日 |
| Contrarian | 逆向投资 | 逆向操作 |
| Trend Following | 趋势跟随 | 跟随趋势 |

### A股交易规则

| 时段 | 时间 | 说明 |
|------|------|------|
| 集合竞价 | 09:15-09:25 | 收集订单，确定开盘价 |
| 上午连续竞价 | 09:30-11:30 | 连续撮合交易 |
| 午间休市 | 11:30-13:00 | 市场暂停 |
| 下午连续竞价 | 13:00-15:00 | 连续撮合交易 |

**涨跌停限制**: 每日价格波动限制为 ±10%

### 10种智能体原型

| 原型 | 初始倾向 | 描述 |
|------|----------|------|
| Trader | Swing, Scalping | 喜欢交易，灵活多变 |
| Gambler | Momentum, Breakout | 高风险偏好 |
| Conservative | Value, Mean Reversion | 稳健保守 |
| Opportunist | Trend Following | 随风倒 |
| Revengeful | Contrarian | 逆向思维 |
| Socialite | 多样化 | 受他人影响 |
| Hermit | 独立探索 | 不受影响 |
| Manipulator | 混合策略 | 策略多变 |
| Learner | 无预设 | 快速学习 |
| Chaotic | 随机 | 不可预测 |

## 🧠 策略探索机制

### 策略选择

智能体根据以下因素选择策略：

1. **置信度** - 历史成功次数影响
2. **期望值** - 策略的预期收益
3. **市场匹配** - 策略与当前市场的契合度
4. **探索概率** - 随机探索新策略

```python
# 策略选择逻辑
scores[strategy] = (
    confidence * (1 + expected_value) * 
    (0.7 + 0.3 * context_match)
)
```

### 风格自适应

交易风格参数会根据交易结果动态调整：

```python
# 亏损后降低风险
if recent_avg < 0:
    risk_tolerance -= 0.02
    position_size -= 0.01

# 波动大时收紧止损
if volatility > 0.03:
    stop_loss -= 0.005
```

### 反馈循环

```
交易决策 → 执行交易 → 记录盈亏 → 更新策略置信度 → 调整风格参数 → 下次决策
```

## 📈 市场机制

### 价格生成

价格由多因素共同决定：

1. **长期趋势**: 周期性波动（约252天一个周期）
2. **中期波动**: 根据市场状态（牛市/熊市/震荡）
3. **短期噪声**: 随机波动
4. **均值回归**: 价格偏离均值时回调

### 订单撮合

采用价格优先、时间优先原则：

```
买单队列: 按价格降序排列
卖单队列: 按价格升序排列

当 最高买价 >= 最低卖价 时成交
```

## 🎯 设计理念

### 为什么叫"永不纳什"？

在博弈论中，纳什均衡是一种稳定状态。但在永不纳什小镇：

- 智能体不断探索新策略
- 风格持续演化
- 市场永远在变化
- 没有固定的最优解

小镇永远不会达到静态均衡，永远充满活力！

### 开放式设计原则

系统遵循"提供空间，不强制行为"的原则：

- ✅ 提供8种策略供探索
- ✅ 提供反馈机制供学习
- ✅ 提供风格参数供调整
- ❌ 不强制智能体使用特定策略
- ❌ 不预设最优交易规则

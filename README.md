# UNASH-TOWN 🏘️

**多智能体博弈小镇模拟系统**

一个包含10个不同初始倾向智能体的博弈模拟系统，在随机变化的环境中进行零和博弈。

## 🎮 项目简介

UNASH-TOWN 是一个多智能体博弈模拟系统，模拟了一个小镇中10个具有不同性格倾向的智能体之间的互动。智能体们在白天进行零和博弈，其他时间进行自由社交活动。

### 核心特性

- **10种不同倾向的智能体**: 每个智能体具有独特的性格特征和行为模式
- **动态环境系统**: 天气、市场状态、特殊事件随机变化
- **零和博弈机制**: 9:00-15:00 进行博弈对决
- **时间系统**: 完整的24小时昼夜循环
- **记忆系统**: 智能体能够记忆过往交互并调整策略

## 📦 安装

```bash
git clone https://github.com/yunxichu/UNASH-TOWN.git
cd UNASH-TOWN
```

项目仅使用 Python 标准库，无需额外依赖。

## 🚀 快速开始

```bash
# 基本运行（默认10个智能体，模拟3天）
python main.py

# 自定义参数
python main.py --agents 10 --days 5 --seed 42

# 静默模式并输出结果到文件
python main.py --days 7 --quiet --output result.json
```

## 📖 详细说明

### 智能体倾向类型

| 倾向 | 英文名 | 特点 |
|------|--------|------|
| 激进型 | aggressive | 高攻击性、低合作、高风险偏好 |
| 保守型 | conservative | 低攻击性、中等合作、低风险偏好 |
| 合作型 | cooperative | 低攻击性、高合作、高信任 |
| 竞争型 | competitive | 高攻击性、低合作、高贪婪 |
| 冒险型 | risk_taking | 中等攻击性、高风险偏好 |
| 规避型 | risk_averse | 低攻击性、中等合作、极低风险偏好 |
| 利他型 | altruistic | 极低攻击性、极高合作、高信任 |
| 自私型 | selfish | 中等攻击性、低合作、极高贪婪 |
| 战略型 | strategic | 均衡型，各特征适中 |
| 随机型 | random | 随机决策 |

### 时间系统

| 时间段 | 阶段 | 活动内容 |
|--------|------|----------|
| 06:00-09:00 | 早晨 | 休息恢复、社交互动 |
| 09:00-15:00 | 博弈时间 | 零和博弈对决 |
| 15:00-18:00 | 下午 | 自由社交活动 |
| 18:00-22:00 | 傍晚 | 状态评估、事件响应 |
| 22:00-06:00 | 夜晚 | 休息恢复能量 |

### 博弈系统

智能体在博弈时可以选择三种行动：

- **合作 (cooperate)**: 寻求双赢局面
- **背叛 (defect)**: 追求个人利益最大化
- **妥协 (compromise)**: 折中方案

博弈收益矩阵（基础值）：

|  | 合作 | 背叛 | 妥协 |
|--|------|------|------|
| **合作** | (3, 3) | (-1, 5) | (2, 2) |
| **背叛** | (5, -1) | (-2, -2) | (3, -1) |
| **妥协** | (2, 2) | (-1, 3) | (1, 1) |

### 环境系统

**天气类型**:
- ☀️ 晴天 - 合作加成，能量消耗降低
- ☁️ 多云 - 正常状态
- 🌧️ 雨天 - 合作降低，能量消耗增加
- ⛈️ 暴风雨 - 显著负面影响
- 🌫️ 大雾 - 信任降低

**市场状态**:
- 📈 繁荣 - 资源倍率1.5x
- ➡️ 稳定 - 资源倍率1.0x
- 📉 萧条 - 资源倍率0.7x
- 💥 危机 - 资源倍率0.5x

**特殊事件**:
- 🎉 节日庆典 - 合作和心情加成
- 🛒 集市日 - 资源和竞争加成
- ⚠️ 灾难事件 - 合作需求增加
- 👥 外来访客 - 信任和资源加成

## 🔧 命令行参数

| 参数 | 说明 | 默认值 |

## 📝 扩展开发

### 添加新的智能体倾向

在 `src/agent.py` 中的 `AgentTendency` 枚举和 `TENDENCY_TRAITS` 字典中添加新类型。

### 自定义博弈规则

修改 `src/game.py` 中的 `PAYOFF_MATRIX` 来调整博弈收益。

### 添加新的环境事件

在 `src/environment.py` 中的 `EventType` 枚举和相关方法中添加新事件。

# 🏘️ 永不纳什小镇 (UNASH-TOWN)

**多智能体生活博弈模拟系统**

一个模拟小镇居民生活的系统，包含睡眠、精力管理、社交互动和股市交易。

## 🎮 项目简介

永不纳什小镇是一个多智能体博弈模拟系统，每个居民都有：
- **8小时睡眠需求** - 睡眠不足会影响第二天精力
- **精力系统** - 每项活动都消耗精力
- **社交生活** - 聊天、交友、建立关系
- **交易能力** - 精力影响交易水平

## 📦 安装

```bash
git clone https://github.com/yunxichu/UNASH-TOWN.git
cd UNASH-TOWN
```

项目仅使用 Python 标准库，无需额外依赖。

## 🚀 快速开始

```bash
# 基本运行
python main.py --days 3 --agents 10

# 自定义参数
python main.py --days 1 --agents 5 --capital 10000 --seed 42

# 静默模式输出到文件
python main.py --days 7 --quiet --output result.json
```

## 📖 核心系统

### 😴 睡眠系统

每个居民需要 **8小时睡眠**：

| 睡眠时长 | 效果 |
|----------|------|
| ≥8小时 | 精力恢复100%，心情愉悦 |
| 6-8小时 | 精力部分恢复，心情一般 |
| <6小时 | 精力不足，累积睡眠债务 |

```
居民睡眠时间示例:
- 小明: 23:00 - 05:00 (6小时) ⚠️ 睡眠不足
- 小红: 21:00 - 05:00 (8小时) ✅ 充足睡眠
- 小华: 21:00 - 07:00 (10小时) ✅ 充足睡眠
```

### ⚡ 精力系统

每项活动消耗精力：

| 活动 | 精力消耗 | 每日上限 |
|------|----------|----------|
| 社交聊天 | 8点 | 15次 |
| 股票交易 | 12点 | 10次 |
| 走路移动 | 3点 | 无限制 |
| 思考决策 | 5点 | 无限制 |
| 休息 | 恢复10点 | 无限制 |

**精力影响交易水平**：

| 精力等级 | 交易技能 |
|----------|----------|
| 满精力 (90-100) | 100% |
| 高精力 (70-89) | 90% |
| 中精力 (40-69) | 70% |
| 低精力 (20-39) | 40% |
| 精疲力尽 (0-19) | 10% |

### 💬 社交系统

居民可以在小镇各处社交：

**地点**：
- 🏛️ 广场 - 公共聚会
- 🍵 茶馆 - 休闲聊天
- 🌳 公园 - 散步交流

**关系类型**：
- 陌生人 → 熟人 → 朋友 → 密友
- 竞争对手 → 敌人

**影响因素**：
- 性格匹配度
- 共同兴趣
- 社交技能（受精力影响）

### 📈 交易系统

**交易时段**：9:00 - 15:00

**交易策略**：
- 价值投资
- 动量交易
- 均值回归
- 技术分析
- 情绪交易
- 随机交易

**市场机制**：
- 订单簿撮合
- 涨跌停限制 (±10%)
- 交易手续费 (0.1%)

## 🎯 设计理念

### 为什么叫"永不纳什"？

在博弈论中，纳什均衡是一种稳定状态。但在我们的小镇里：
- 居民不断学习、社交、成长
- 环境持续变化
- 关系动态发展

小镇永远不会达到静态均衡，永远充满活力！

### 核心机制

```
睡眠 → 精力 → 活动 → 社交/交易 → 心情 → 睡眠质量
  ↑                                              ↓
  └──────────────── 循环往复 ────────────────────┘
```

## 🔌 可扩展性

系统支持动态添加智能体和API接入：

```python
from src.agent_manager import AgentManager
from src.api_server import APIServer

# 创建管理器
manager = AgentManager(max_agents=1000)

# 启动API服务
api = APIServer(manager, port=8080)
api.start()
```

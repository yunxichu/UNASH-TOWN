# 🏘️ 永不纳什小镇 (UNASH-TOWN)

**多智能体社会博弈模拟系统**

一个模拟小镇居民生活的系统，每个居民都有独特的人格类型，通过博弈、社交、联盟形成复杂的社会网络。

## 🎮 核心特性

### 10种智能体类型

| 类型 | 特征 | 描述 |
|------|------|------|
| Trader | 喜欢交换信息 | 建立贸易网络，| Gambler | 高风险策略 | 敢于冒险 |
| Conservative | 稳健策略 | 避免风险 |
| Opportunist | 看谁强就靠谁 | 随风倒 |
| Revengeful | 记仇 | 有仇必报 |
| Socialite | 频繁交流 | 建立广泛关系 |
| Hermit | 不太社交 | 独来独往 |
| Manipulator | 散布假消息 | 操控他人 |
| Learner | 快速学习 | 不断调整策略 |
| Chaotic | 行为随机 | 不可预测 |

### 人格参数系统

```
risk        风险偏好 (0-1)
trust       信任倾向 (0-1)
memory      记忆长度 (0-1)
greed       贪婪程度 (0-1)
revenge     报复倾向 (0-1)
social      社交活跃度 (0-1)
learning    学习速度 (0-1)
```

### 博弈系统 - 囚徒困境

|  | 合作 | 背叛 |
|------|------|------|
| 合作 | (3, 3) | (0, 5) |
| 背叛 | (5, 0) | (1, 1) |

**涌现行为**:
- 合作 → 双赢
- 背叛 → 单方获利
- 长期信任/背叛循环
- 复仇机制

### 社交系统

- **信息传播**: 分享真实/虚假信息
- **联盟形成**: 建立合作关系
- **声誉系统**: 信任评分
- **承诺机制**: 履行/违背承诺

### 精力系统

- 每项活动消耗精力
- 篝力影响交易技能
- 睡眠不足影响第二天精力

## 🚀 快速开始

```bash
python main.py --days 3 --agents 10
```

## 🎯 设计理念

**为什么叫"永不纳什"？**

在博弈论中，纳什均衡是一种稳定状态。但在永不纳什小镇：
- 居民不断学习、社交、成长
- 关系动态发展
- 环境持续变化

小镇永远不会达到静态均衡，永远充满活力！
